# Brass-Player

🎺 吹奏楽の練習をもっとスムーズにするためのシンプルなWebサイトです。

## 概要
**Brass-Player** は、練習番号のボタンを押すだけで、  
あらかじめ埋め込まれた動画の該当位置へジャンプできるWebサイトです。

吹奏楽部での個人練習や、身内向けの共有用途を想定しています。

## 使用技術
- HTML  
- CSS  
- JavaScript  

特別なライブラリやフレームワークは使用していません。

## 使い方
1. GitHub Pages にアクセスします  
2. 表示された練習番号のボタンを押します  
3. 対応する練習動画の位置へ自動でジャンプします  

※ コードを書く必要はありません。

## インストール
このプロジェクトは **GitHub Pages** でそのまま使用できます。  
ローカル環境へのインストール作業は不要です。

## 想定用途
- 自主練習時の確認用
- パート練習での共有
- 身内・部内向けの簡易ツール
